# 로컬 개발 환경 설정

## 사전 요구사항

- Docker
- Docker Compose
- pnpm

## 환경 구성

프로젝트는 다음과 같은 서비스들로 구성되어 있습니다:

### PostgreSQL 복제 구성

- 메인 데이터베이스 (Master)

  - 포트: 15432
  - 사용자: user
  - 비밀번호: password
  - 복제 사용자: repl
  - 복제 비밀번호: password

- 대기 데이터베이스 (Slave)
  - 포트: 15433
  - 메인 DB와 동일한 인증 정보 사용
  - 자동 복제 설정됨

### Redis 캐시 서버

- 포트: 16379
- 영구 데이터 저장을 위한 볼륨 마운트 설정

## 데이터베이스 권한

복제 사용자(repl)는 다음 권한을 가집니다:

- public 스키마의 모든 테이블에 대한 모든 권한
- SUPERUSER 권한
